Internet Computer tiene algunos puntos que lo hacen resaltar de otros ecosistemas, mismos que resultan en una ventaja al momento de lanzar software enfocado en blockchain como:

  • Almacenamiento on-chain
  • Usuarios no necesitan de una wallet o token para usar DApps.
  • Tanto frontend como backend se pueden desplegar en la cadena de IC mediante Canisters.
  • Los canisters representan una evolución de los smart contracts y se ejecutan dentro de un ambiente basado en WASM.

  • Internet Computer es muy versátil en el desarrollo, con tener conocimientos básicos de algún lenguaje como JS y conocimiento general de informática se puede iniciar a desarrollar en web3 con IC.

    • Puede utilizarse cualquier lenguaje con soporte para compilar en formato .wasm (bytecode). Los lenguajes que cuentan oficialmente con un CDK (Canister Development Kit) son:

  • Motoko
  • Rust
  • Javascript/TypeScript
  • Python

  • Es un lenguaje de programación desarrollado exclusivamente para la blockchain de Internet Computer. Es un lenguaje potente que recuerda a JavaScript y Rust por su sintaxis. Es sencillo de aprender y cuenta con una gran documentación detrás.

  • Si, actualmente Internet Computer cuenta con una EVM llamada bitfinity además de tener también como protocolo una integración nativa con Ethereum para desarrollar sobre este ecosistema.
